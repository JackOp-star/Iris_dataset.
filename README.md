# Iris Dataset Classification Project

This repository contains a Machine Learning workflow designed to predict the species of an Iris flower based on its morphological features. The project utilizes the classic Iris Dataset obtained from Kaggle.

---

## 📋 Project Overview

The goal of this project is to build and evaluate a predictive model that accurately classifies Iris flowers into one of three species:
* *Iris-setosa*
* *Iris-versicolor*
* *Iris-virginica*

The dataset includes four features for each sample: sepal length, sepal width, petal length, and petal width (all measured in centimeters).

---

## 📁 Repository Structure

* `iris-train.xlsx`: The training dataset used to fit the model.
* `iris-test.xlsx`: The testing dataset used to evaluate model performance.
* `iris_data_project.ipynb`: Jupyter Notebook containing data preprocessing, exploratory data analysis (EDA), model training, and evaluation.
* `README.md`: Project documentation.

---

## 🛠️ Workflow & Methodology

1.  **Data Cleaning & Preprocessing:** * Handled missing values and verified data types.
    * Formatted column names and cleaned string entries for consistency.
2.  **Exploratory Data Analysis (EDA):** Analyzed feature distributions and correlations to understand relationships between flower measurements and their species.
3.  **Model Training:** Trained a classification model using Python and popular data science libraries (such as Pandas, NumPy, and Scikit-Learn).
4.  **Evaluation:** Tested model performance on unseen data using standard classification metrics.
